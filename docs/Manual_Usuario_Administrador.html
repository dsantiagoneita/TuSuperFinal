1. Introduccion

Bienvenido al manual del Administrador de TuSuper. Como administrador, tienes el nivel mas alto de acceso al sistema y eres responsable de la gestion global de la plataforma.

1.1 Responsabilidades del Administrador

  • Supervision general del sistema
  • Gestion de todos los usuarios (clientes, tenderos, administradores)
  • Creacion de cuentas de tenderos
  • Monitoreo de productos y pedidos
  • Generacion y analisis de reportes
  • Activacion y desactivacion de cuentas
  • Mantenimiento de la integridad del sistema
Advertencia: El rol de administrador tiene acceso total al sistema. Usa estas funciones con responsabilidad y cuidado.

1.2 Diferencias con Otros Roles

Funcion Cliente Tendero Admin
Ver productos Si Si Si
Gestionar productos No Si Ver
Realizar pedidos Si No No
Gestionar pedidos Ver propios Si Ver todos
Gestionar usuarios No No Si
Crear tenderos No No Si
Ver reportes globales No Parcial Si

2. Acceso al Sistema

2.1 Credenciales de Administrador

Las credenciales de administrador son configuradas durante la instalacion del sistema o proporcionadas por el administrador tecnico.

Credenciales de prueba:
Email: admin@tusuper.com
Contrasena: admin123

2.2 Inicio de Sesion

  1. Accede a la pagina de login de TuSuper
  2. Ingresa tu email de administrador
  3. Ingresa tu contrasena
  4. Haz clic en "Iniciar Sesion"
  5. Seras redirigido al Panel de Administracion

2.3 Estructura de la Interfaz

La interfaz administrativa consta de:

  • Barra de navegacion superior: Logo, titulo y menu de usuario
  • Menu lateral (Sidebar): Navegacion entre modulos
  • Area principal: Contenido del modulo seleccionado

2.4 Menu Lateral del Administrador

4. Gestion de Usuarios

Modulo central para administrar todos los usuarios del sistema.

4.1 Acceso

Haz clic en "Usuarios" en el menu lateral.

4.2 Filtrar Usuarios por Rol

Usa el selector de filtro para ver usuarios especificos:

  • Todos los roles: Muestra todos los usuarios
  • Clientes: Solo usuarios con rol USUARIO
  • Tenderos: Solo usuarios con rol TENDERO
  • Administradores: Solo usuarios con rol ADMIN

4.3 Informacion de Usuarios

La tabla muestra:

Columna Descripcion
ID Identificador unico del usuario
Nombre Nombre completo
Email Correo electronico (usuario de acceso)
Telefono Numero de contacto
Roles Badges indicando los roles asignados
Estado Activo (verde) o Inactivo (gris)
Acciones Botones de gestion

4.4 Acciones sobre Usuarios

Ver Detalle del Usuario

  1. Haz clic en el icono de ojo
  2. Se mostrara informacion completa del usuario
  3. Podras ver historial y detalles adicionales

Desactivar Usuario

  1. En usuarios activos, haz clic en el boton X roja
  2. El usuario sera desactivado inmediatamente
  3. El usuario no podra iniciar sesion hasta ser reactivado
Precaucion: Desactivar un usuario no elimina sus datos ni historial. El usuario puede ser reactivado posteriormente.

Activar Usuario

  1. En usuarios inactivos, haz clic en el boton check verde
  2. El usuario sera reactivado inmediatamente
  3. El usuario podra iniciar sesion nuevamente

5. Creacion de Tenderos

Solo los administradores pueden crear cuentas de tendero.

5.1 Acceso al Formulario

  1. Ve a la seccion de Usuarios
  2. Haz clic en el boton "Nuevo Tendero" (esquina superior derecha)

5.2 Completar el Formulario

Campos requeridos (marcados con *):

Campo Descripcion Requerido
Nombre Completo Nombre y apellido del tendero Si
Email Correo electronico (sera su usuario) Si
Telefono Numero de contacto No
Direccion Direccion del tendero No
Contrasena Contrasena inicial (minimo 6 caracteres) Si
Confirmar Contrasena Repetir la contrasena Si

5.3 Crear el Tendero

  1. Completa todos los campos requeridos
  2. Verifica que las contrasenas coincidan
  3. Haz clic en "Crear Tendero"
  4. El tendero recibira un mensaje de confirmacion
  5. Proporciona las credenciales al nuevo tendero de forma segura
Exito: Una vez creado, el tendero podra acceder inmediatamente al sistema con las credenciales proporcionadas.
Seguridad: Comunica las credenciales de forma segura y recomienda al tendero cambiar su contrasena inicial.

6. Supervision de Productos

Vista de solo lectura del catalogo completo de productos.

6.1 Informacion Visible

Columna Descripcion
ID Identificador unico del producto
Codigo Codigo de barras
Producto Nombre y marca
Categoria Clasificacion del producto
Precio Precio de venta en COP
Stock Cantidad disponible (rojo si bajo)
Estado Activo o Inactivo
Nota: La gestion de productos (crear, editar, eliminar) es responsabilidad del tendero. El administrador solo supervisa.

6.2 Identificar Problemas

Como administrador, debes estar atento a:

  • Productos con stock bajo: Coordinar con tenderos para reabastecimiento
  • Productos inactivos: Verificar si deben reactivarse o eliminarse
  • Precios inconsistentes: Revisar con tenderos

7. Supervision de Pedidos

Vista completa de todos los pedidos del sistema.

7.1 Filtrar Pedidos

Usa el filtro de estado para ver pedidos especificos:

  • Todos los estados
  • PENDIENTE: Esperando confirmacion
  • CONFIRMADO: Confirmados por el tendero
  • EN_PREPARACION: Siendo preparados
  • ENVIADO: En camino al cliente
  • ENTREGADO: Completados exitosamente
  • CANCELADO: Pedidos cancelados

7.2 Informacion del Pedido

Columna Descripcion
Pedido Numero unico (ej: PED-001)
Fecha Fecha y hora del pedido
Cliente Nombre del comprador
Items Cantidad de productos
Total Valor total en COP
Estado Estado actual del pedido

7.3 Ver Detalle

Haz clic en el boton de ojo para ver:

  • Informacion completa del cliente
  • Lista detallada de productos
  • Direccion de entrega
  • Observaciones del cliente
  • Historial de cambios de estado
Nota: La gestion de pedidos (cambiar estados) es responsabilidad del tendero. El administrador supervisa el proceso.

8. Reportes del Sistema

Seccion de analisis y metricas globales.

8.1 Metricas de Usuarios

  • Total Usuarios: Cantidad total registrada
  • Clientes: Usuarios con rol USUARIO
  • Tenderos: Usuarios con rol TENDERO

8.2 Metricas de Productos

  • Total Productos: Productos en catalogo
  • Productos Activos: Disponibles para venta

8.3 Metricas de Pedidos

  • Total Pedidos: Todos los pedidos realizados
  • Pedidos Entregados: Completados exitosamente

8.4 Metricas Financieras

Total en Ventas: Suma de todos los pedidos con estado ENTREGADO, mostrado en pesos colombianos (COP).

8.5 Analisis Recomendado

Usa estos reportes para:

  • Evaluar el crecimiento de la plataforma
  • Identificar tendencias en ventas
  • Medir la tasa de conversion de pedidos
  • Planificar estrategias de negocio

9. Seguridad y Buenas Practicas

9.1 Gestion de Contrasenas

  • Usa contrasenas fuertes (minimo 8 caracteres, combinando letras, numeros y simbolos)
  • No compartas tu contrasena de administrador
  • Cambia la contrasena periodicamente
  • Al crear tenderos, usa contrasenas temporales que deban cambiar

9.2 Gestion de Sesiones

  • Siempre cierra sesion al terminar de trabajar
  • No dejes sesiones abiertas en computadores publicos
  • El sistema cierra sesiones automaticamente despues de inactividad

9.3 Gestion de Usuarios

Mejores practicas:
  • Desactiva usuarios sospechosos inmediatamente
  • Revisa periodicamente la lista de usuarios activos
  • Verifica que solo las personas autorizadas tengan rol de tendero o admin
  • Mant√©n un registro de cambios importantes

9.4 Monitoreo Regular

Se recomienda revisar diariamente:

  • Dashboard para metricas generales
  • Pedidos pendientes o atrasados
  • Nuevos usuarios registrados
  • Productos con stock critico

10. Solucion de Problemas

10.1 Problemas de Acceso

Problema Solucion
No puedo iniciar sesion Verifica email y contrasena. Contacta al administrador tecnico si persiste.
Sesion expira muy rapido Es normal por seguridad. Inicia sesion nuevamente.
No veo el menu de administrador Verifica que tu cuenta tenga rol ADMIN asignado.

10.2 Problemas con Usuarios

Problema Solucion
Usuario no puede acceder Verifica que este activo. Si esta inactivo, reactivalo.
Tendero olvido su contrasena Contacta al administrador tecnico para resetearla.
Email duplicado al crear usuario El email ya existe. Usa otro email o busca el usuario existente.

10.3 Problemas con Pedidos

Problema Solucion
Pedido atascado en un estado Contacta al tendero para que actualice el estado.
Cliente quiere cancelar pedido Coordina con el tendero para cambiar estado a CANCELADO.

10.4 Problemas Tecnicos

Problema Solucion
Pagina no carga Refresca la pagina. Verifica conexion a internet.
Datos no se actualizan Limpia cache del navegador y refresca.
Error inesperado Toma captura de pantalla y contacta soporte tecnico.

10.5 Contacto de Soporte

Para problemas tecnicos que no puedas resolver, contacta al equipo de desarrollo o soporte tecnico con:

  • Descripcion detallada del problema
  • Capturas de pantalla si es posible
  • Pasos para reproducir el error
  • Fecha y hora aproximada del incidente
