1. Introduccion

Bienvenido al manual del Tendero de TuSuper. Como tendero, eres el encargado de gestionar los productos, el inventario y los pedidos de la tienda.

Tus principales responsabilidades incluyen:

  • Administrar el catalogo de productos
  • Mantener actualizado el inventario
  • Procesar y gestionar pedidos de clientes
  • Controlar el stock y reabastecer productos
  • Utilizar el escaner de codigo de barras para busqueda rapida
Nota: Tu cuenta de tendero es creada por el administrador del sistema. Si no tienes acceso, contacta al administrador.

2. Acceso al Sistema

2.1 Inicio de Sesion

  1. Ingresa a la pagina de login de TuSuper
  2. Introduce el email proporcionado por el administrador
  3. Introduce tu contrasena
  4. Haz clic en "Iniciar Sesion"
  5. Seras redirigido automaticamente al panel de tendero

2.2 Estructura de la Interfaz

La interfaz del tendero consta de:

  • Barra de navegacion superior: Acceso rapido y menu de usuario
  • Menu lateral (Sidebar): Navegacion entre secciones
  • Area de contenido: Zona principal de trabajo

2.3 Menu Lateral

Opcion Descripcion

Productos Gestion completa del catalogo
Pedidos Ver y gestionar pedidos de clientes
Inventario Control de stock y alertas

4. Gestion de Productos

4.1 Ver Lista de Productos

Accede a Productos en el menu lateral para ver el catalogo completo.

La tabla muestra:

  • Codigo: Codigo de barras del producto
  • Producto: Nombre y marca
  • Categoria: Clasificacion del producto
  • Precio: Precio de venta en COP
  • Stock: Cantidad disponible (resaltado en rojo si es bajo)
  • Estado: Activo o Inactivo
  • Acciones: Editar, ajustar stock, activar/desactivar

4.2 Filtrar por Categoria

Usa los botones de categoria en la parte superior para filtrar productos rapidamente.

4.3 Crear Nuevo Producto

  1. Haz clic en el boton "Nuevo Producto"
  2. Completa el formulario:
    • Nombre del Producto *: Nombre descriptivo
    • Codigo de Barras: Escanea o escribe el codigo
    • Descripcion: Detalles adicionales
    • Precio *: Precio en COP
    • Cantidad *: Stock inicial
    • Stock Minimo: Alerta cuando baje de este numero
    • Categoria: Selecciona o escribe nueva
    • Marca: Fabricante del producto
    • URL de Imagen: Enlace a imagen del producto
    • Producto Activo: Si estara visible para clientes
  3. Haz clic en "Guardar Producto"
Consejo: Los campos marcados con * son obligatorios.

4.4 Editar Producto

  1. En la lista de productos, haz clic en el icono de lapiz
  2. Modifica los campos necesarios
  3. Haz clic en "Guardar Producto"

4.5 Ajustar Stock Rapido

  1. Haz clic en el icono de +/- junto al producto
  2. Se abrira un modal mostrando el stock actual
  3. Ingresa la cantidad a agregar (positivo) o quitar (negativo)
  4. Haz clic en "Actualizar Stock"

4.6 Activar/Desactivar Producto

  • Desactivar: Haz clic en el icono de X roja - El producto no sera visible para clientes
  • Activar: Haz clic en el icono de check verde - El producto volvera a estar disponible
Nota: Desactivar un producto no elimina su historial ni lo borra del sistema.

5. Control de Inventario

5.1 Vista General

La seccion de inventario muestra tres tarjetas resumen:

  • Total Productos: Todos los productos en el sistema
  • Productos Activos: Disponibles para venta
  • Stock Bajo: Productos que necesitan atencion

5.2 Productos con Stock Bajo

Tabla especial (resaltada en rojo) que muestra:

  • Nombre del producto
  • Stock actual
  • Stock minimo configurado
  • Boton para reabastecer

5.3 Lista Completa de Inventario

Tabla con todos los productos y sus niveles de stock:

  • Productos con stock bajo se resaltan en amarillo
  • Estados: Agotado, Bajo, OK
Recomendacion: Revisa esta seccion al inicio y final de cada dia para mantener el inventario actualizado.

6. Escaner de Codigo de Barras

Herramienta poderosa para busqueda rapida de productos usando un lector USB.

6.1 Requisitos

  • Lector de codigo de barras USB conectado al computador
  • El lector debe estar configurado para enviar Enter al final del codigo

6.2 Uso con Lector USB

  1. Accede a la seccion "Escaner"
  2. Asegurate que el campo de texto tenga el foco (cursor parpadeando)
  3. Apunta el lector al codigo de barras del producto
  4. El sistema detectara automaticamente el codigo
  5. Se mostrara la informacion del producto encontrado

6.3 Busqueda Manual

  1. Escribe el codigo de barras en el campo de texto
  2. Haz clic en "Buscar" o presiona Enter

6.4 Resultado de Busqueda

Si el producto existe, veras:

  • Nombre y descripcion
  • Codigo de barras
  • Categoria y marca
  • Precio
  • Stock actual con indicador de estado
  • Botones para editar o escanear otro

6.5 Ajuste Rapido de Stock desde Escaner

Cuando encuentras un producto, puedes ajustar el stock directamente:

  • Boton -: Reduce el stock
  • Boton +: Aumenta el stock
  • Campo numerico: Cantidad a ajustar

6.6 Producto No Encontrado

Si el codigo no existe en el sistema:

  • Se mostrara un mensaje de "Producto No Encontrado"
  • Opcion para Registrar Nuevo Producto con ese codigo

6.7 Historial de Escaneos

La parte inferior muestra los ultimos escaneos realizados con:

  • Hora del escaneo
  • Codigo escaneado
  • Nombre del producto (si se encontro)
  • Estado (Encontrado/No encontrado)

7. Gestion de Pedidos

7.1 Ver Pedidos

Accede a Pedidos para ver todos los pedidos del sistema.

7.2 Filtrar Pedidos

Usa el filtro de estado para ver pedidos especificos:

  • Todos los estados
  • PENDIENTE
  • CONFIRMADO
  • EN_PREPARACION
  • ENVIADO
  • ENTREGADO
  • CANCELADO

7.3 Informacion del Pedido

La tabla muestra:

  • Pedido: Numero unico (ej: PED-001)
  • Fecha: Cuando se realizo
  • Cliente: Nombre del comprador
  • Direccion: Donde entregar
  • Items: Cantidad de productos
  • Total: Valor total del pedido
  • Estado: Estado actual

7.4 Gestionar un Pedido

  1. Haz clic en el boton de ojo para ver detalles
  2. En la vista de detalle puedes:
    • Ver todos los productos del pedido
    • Ver observaciones del cliente
    • Cambiar el estado del pedido

7.5 Flujo de Estados del Pedido

Estado Actual Accion Siguiente Estado
PENDIENTE Confirmar pedido CONFIRMADO
CONFIRMADO Iniciar preparacion EN_PREPARACION
EN_PREPARACION Marcar como enviado ENVIADO
ENVIADO Confirmar entrega ENTREGADO
Cualquier estado Cancelar CANCELADO
Importante: Al cambiar un pedido a ENTREGADO, este se considerara completado y contara en las ventas.

9. Mejores Practicas

9.1 Rutina Diaria Recomendada

  1. Inicio del dia:
    • Revisar pedidos pendientes
    • Verificar alertas de stock bajo
  2. Durante el dia:
    • Procesar pedidos a medida que llegan
    • Actualizar stock cuando lleguen productos
    • Usar el escaner para verificaciones rapidas
  3. Fin del dia:
    • Verificar que no queden pedidos sin procesar
    • Revisar inventario
    • Planificar reabastecimiento

9.2 Consejos para el Inventario

  • Configura el stock minimo de forma realista segun la rotacion del producto
  • Actualiza el stock inmediatamente al recibir mercancia
  • Usa el escaner para conteos rapidos
  • Desactiva productos descontinuados en lugar de eliminarlos

9.3 Consejos para Pedidos

  • Procesa los pedidos en orden de llegada (FIFO)
  • Actualiza el estado del pedido en cada etapa
  • Lee las observaciones del cliente antes de preparar
  • Verifica la direccion antes de enviar
